Coop CVI

Frontend framework for Coop's corporate visual identity

Looking for Github repo and styleguide? It's here.

Intro

This project containts basic styling for Coop-themed websites. It combines a subset of Bootstrap 4 variables and general styling with specific Coop styling based on the corporate visual identity (CVI).

Coop styles use BEM syntax, and class names are prefixed coop- to avoid name clashing.

The project can be used as prebuild CSS or customized with additional Bootstrap elements by tweaking the Sass source files.

Add the following stylesheet link to include the prebuilt styles. 

          
            <link href="https://cdn.jsdelivr.net/gh/coop-digital/coop-cvi@v1.0.0-beta/dist/styles.css" rel="stylesheet">

Source code including SASS-variables is located here.

Fonts

COOP font is owned and produced by Coop. It has 5 weights:

  • COOP Light (300)
  • COOP Regular (400)
  • COOP Medium (500)
  • COOP Semibold (600)
  • COOP Bold (700)

Font is included in this project and added to body by default.

          div {
            font-family: COOP;
          }

          div {
            font-family: $font-family-base;
          }

              <script>
                /* Password toggle script. Use as is or integrate in your JS-framework. */

                function togglePasswordVisibility() {
                  var passwordInput = document.getElementById('coop-password-input')
                  var toggleIcon = document.getElementById('coop-password-input__icon')

                  if (passwordInput.type === 'text') {
                    passwordInput.type = 'password'
                    toggleIcon.classList.add('coop-icon-eye')
                    toggleIcon.classList.remove('coop-icon-eye-off')
                  } else {
                    passwordInput.type = 'text'
                    toggleIcon.classList.remove('coop-icon-eye')
                    toggleIcon.classList.add('coop-icon-eye-off')
                  }
                }
              </script>

              CDN base path:
              https://cdn.jsdelivr.net/gh/coop-digital/coop-cvi@v1.0.0-beta/dist/

              CDN + Github to generate path for coop-red.svg:
              https://cdn.jsdelivr.net/gh/coop-digital/coop-cvi@v1.0.0-beta/dist/logo/coop/coop-red.svg

Navigation

Coop navigation has three levels: global, site and context.

Global nav should contain links that go accross Coop sites, such as signup, login and customer service contact

Site nav contains the most important site specific links and features, such as search

Context nav is the traditional site specific page navigation
